## Что это?

Музыкальный веб-сайт с подборками рок-музыки, карточками жанров, плейлистами и треками.  
Сделано в минималистичном тёмном дизайне с акцентами на красный и жёлтый цвета.

Основные разделы:
- Главная — обзор жанров и энергичное приветствие
- Жанры (рок, хард-рок, альтернатив, панк/метал и др.)
- Треки (пример: Limp Bizkit, Linkin Park и т.п.)
- Статические страницы (статьи, плейлисты и т.д.)

## Технический стек

- Backend: Django 6.x
- Frontend: HTML + CSS
- База данных: SQLite (для разработки)
- Парсинг треков: собственный скрипт (parse_hitmos)

## Как запустить локально

1. Клонируйте репозиторий
```bash
git clone https://github.com/JoyHubN/djng.git
```

2. Перейдите в папку проекта
```bash
cd djng
```

3. Создайте виртуальное окружение

```bash
python -m venv .venv
source .venv/bin/activate    # Linux/Mac
.venv\Scripts\activate       # Windows
```

5. Установить зависимости
```bash
pip install -r requirements.txt
```

6. Примените миграции и создайте суперпользователя

```bash
python manage.py migrate
python manage.py createsuperuser
```

7. Запустите
```bash
make run
```
